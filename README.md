# Netflix India Clone (2024)

A clone of the Netflix India landing page built with HTML, CSS and vanilla JavaScript.

##  Live Demo
[Click here to view](https://swapnil-2004.github.io/Netflix_Clone-2024/)

##  Built With
- HTML
- CSS (Flexbox, Grid, Responsive Design)
- JavaScript (FAQ Accordion)

##  Features
- Responsive design (mobile + desktop)
- Hero section with background image
- Working FAQ accordion (one open at a time)
- Smooth scroll
- Local assets (no external CDN dependency)
- Video animations inside device mockups

##  Project Structure
```
Netflix_Clone/
├── assets/
│   ├── images/
│   └── videos/
├── index.html
└── style.css
```
